Simulação 2D

Teoria das Filas
Descrição do Sistema

Simulação de filas de um supermercado. O modelo simula o comportamento de filas em um supermercado.

Variáveis estocásticas

1. Tempo de setup da máquina; 2. Dimensão das peças; 3. Tempo de processamento das peças; 4. Quantidade de peças com defeito ao fim do processamento;

Ferramentas utilizadas

O projeto será desenvolvido na plataforma IOS, com o framework SceneKit.
